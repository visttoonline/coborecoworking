# Guia de Fotos — COBORE Site

## Como usar

1. As fotos Bertuol já estão em `cobore-site/public/img/`
2. Renomeie cada foto conforme a tabela abaixo (ou rode o script no Terminal)
3. Pronto — o site carrega tudo automaticamente
